# R17 — Use of Oblique Symbol

- **ID:** R17
- **Category (quality focus):** Non-ambiguity
- **Scope:** individual requirement
- **Detector type:** deterministic
- **Source:** INCOSE GtWR v4 (INCOSE-TP-2010-006-04, June 2023)

## Rule
Avoid the use of the oblique ("/") symbol except in units (e.g. Km/hr) or fractions.

## Trigger terms

```
/
```
