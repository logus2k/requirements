# R33 — Range of Values

- **ID:** R33
- **Category (quality focus):** Tolerance
- **Scope:** individual requirement
- **Detector type:** semantic
- **Source:** INCOSE GtWR v4 (INCOSE-TP-2010-006-04, June 2023)

## Rule
Define each quantity with a range of values appropriate to the entity to which the quantity applies and against which the entity will be verified or validated.
