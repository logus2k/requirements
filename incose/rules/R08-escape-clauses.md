# R8 — Escape Clauses

- **ID:** R8
- **Category (quality focus):** Accuracy
- **Scope:** individual requirement
- **Detector type:** deterministic
- **Source:** INCOSE GtWR v4 (INCOSE-TP-2010-006-04, June 2023)

## Rule
Avoid the inclusion of escape clauses that state vague conditions or possibilities.

## Trigger terms

```
so far as is possible, as little as possible, where possible, as much as possible, if it should prove necessary, if necessary, to the extent necessary, as appropriate, as required, to the extent practical, if practicable
```
