# R20 — Purpose Phrases

- **ID:** R20
- **Category (quality focus):** Singularity
- **Scope:** individual requirement
- **Detector type:** deterministic
- **Source:** INCOSE GtWR v4 (INCOSE-TP-2010-006-04, June 2023)

## Rule
Avoid phrases that indicate the "purpose of", "intent of", or "reason for" the requirement statement.

## Trigger terms

```
purpose of, intent of, reason for, in order to, so that, so as to
```

## Detector notes
Detectable by matching the trigger terms / symbol below — exact, cites the offending token. No LLM needed.
