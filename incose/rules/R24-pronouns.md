# R24 — Pronouns

- **ID:** R24
- **Category (quality focus):** Completeness
- **Scope:** individual requirement
- **Detector type:** deterministic
- **Source:** INCOSE GtWR v4 (INCOSE-TP-2010-006-04, June 2023)

## Rule
Avoid the use of personal and indefinite pronouns.

## Trigger terms

```
it, its, they, them, their, this, that, these, those, he, she, his, her, we, our, you, your, anyone, everyone, someone, anything, everything, something
```

## Detector notes
Detectable by matching the trigger terms / symbol below — exact, cites the offending token. No LLM needed.
